# ⚡ Быстрый старт (5 минут)

## Шаг 1: Скачайте Node.js

Если не установлен:
→ https://nodejs.org/ (выбирайте LTS версию)

Проверьте установку:
```bash
node --version
npm --version
```

Должны появиться версии вроде: `v18.0.0`

## Шаг 2: Скачайте файлы проекта

Все файлы находятся в этой папке. Скопируйте их в одну папку, например:
```
C:\Users\YourName\fomo-digest\
```

## Шаг 3: Установите зависимости

Откройте терминал в папке проекта и выполните:

```bash
npm install
```

Это установит нужные библиотеки (займёт ~30 секунд).

## Шаг 4: Запустите первый дайджест

```bash
npm start
```

Вы увидите:
```
============================================================
   FOMO-утро: Генератор дайджеста AI новостей
============================================================

📥 Загружаю Every.to...
✅ Every.to: найдено 8 статей
📥 Загружаю DeepLearning.AI Batch...
✅ DeepLearning.AI Batch: найдено 5 статей
...
```

Это займёт 30-60 секунд (зависит от интернета).

Когда готово:
```
📊 Отобрано топ 10 статей из 45

✅ Дайджест сохранён: /path/to/magazines/2024-04-21.html
🌐 Откройте в браузере: file:///path/to/magazines/2024-04-21.html

============================================================
✨ Дайджест готов!
============================================================
```

## Шаг 5: Откройте дайджест

Скопируйте путь из вывода выше и откройте в браузере.

Или просто откройте папку `magazines/` и кликните на HTML файл.

## Красивая картинка! ✨

Поздравляем, ваш первый дайджест готов!

## Что дальше?

### ✅ Хотите генерировать каждый день в 10:00?

```bash
npm run schedule
```

Оставьте это окно открытым. Дайджест будет создаваться автоматически каждое утро.

```
🚀 Планировщик запущен
📅 Дайджесты будут генерироваться каждый день в 10:00 (Москва)
⏹️  Для остановки нажмите Ctrl+C
```

### ✅ Хотите запускать в фоне на Windows?

Используйте **NSSM** (NonSugaringSysem Manager):

1. Скачайте NSSM: http://nssm.cc/download
2. Распакуйте в `C:\nssm\`
3. Откройте CMD и введите:

```batch
C:\nssm\nssm.exe install fomo-digest "C:\Program Files\nodejs\node.exe" "C:\path\to\schedule.js"
C:\nssm\nssm.exe start fomo-digest
```

Готово! Теперь дайджесты будут генерироваться каждый день в 10:00, даже если вы выключите компьютер и включите его снова.

### ✅ Хотите запускать в фоне на Mac/Linux?

Используйте PM2:

```bash
npm install -g pm2
pm2 start schedule.js --name "fomo-digest"
pm2 startup
pm2 save
```

Готово! PM2 будет автоматически перезапускать дайджест при перезагрузке.

## Если что-то не работает

### ❌ "command not found: npm"

Переустановите Node.js https://nodejs.org/

### ❌ "Cannot find module"

```bash
npm install
```

### ❌ "No articles found"

Проверьте интернет и попробуйте ещё раз.

### ❌ Файл не создаётся

Проверьте права доступа к папке `magazines/` или создайте её вручную.

## Файлы дайджестов

Все дайджесты сохраняются в папке `magazines/` с именем `YYYY-MM-DD.html`:

```
magazines/
├── 2024-04-20.html  (от 20 апреля)
├── 2024-04-21.html  (от 21 апреля)
└── 2024-04-22.html  (от 22 апреля)
```

Каждый файл - это полностью самостоятельный HTML, который работает без интернета.

Вы можете:
- 📧 Отправить кому-нибудь по почте
- ☁️ Загрузить на сервер
- 🔄 Поделиться ссылкой
