# FOMO-утро 🌅

Ежедневный дайджест лучших статей об AI, инструментах и программировании.

Каждое утро в **10:00 по Москве** скрипт собирает топ-10 статей с 6 авторитетных источников и оформляет их в красивый HTML-журнал.

## Особенности

✨ **Красивый дизайн**
- 10 разных стилей оформления для каждой статьи
- Встроенные изображения
- Отзывчивый дизайн (работает на телефоне)
- Google Fonts (Fraunces + Inter)

📚 **Умный парсинг**
- Автоматическое обновление каждое утро
- Релевантность по ключевым словам (AI, tools, privacy, science и т.д.)
- Обработка ошибок при загрузке

🎯 **Простота в использовании**
- Локальное сохранение (никакого облака)
- Один скрипт для запуска
- Можно легко расширить на сервер позже

## Источники

1. **Every.to** - глубокие статьи про AI и технологии
2. **DeepLearning.AI Batch** - новости в ML
3. **The Rundown AI** - подборка AI новостей
4. **The Zvi** - аналитика и прогнозы
5. **Jack Clark** - исследования и тренды
6. **Jeff Bullas JABS** - tips для маркетологов и бизнеса

## Установка

### 1. Требования

- **Node.js** 14+ (скачайте с https://nodejs.org/)
- Интернет-соединение
- ~100 МБ места на диске

### 2. Клонирование / Скачивание

```bash
# Если у вас Git:
git clone https://github.com/your-username/fomo-digest.git
cd fomo-digest

# Или просто скачайте файлы в папку fomo-digest/
```

### 3. Установка зависимостей

```bash
npm install
```

Это установит:
- `axios` - для загрузки веб-страниц
- `cheerio` - для парсинга HTML
- `node-cron` - для планирования задач
- `dotenv` - для конфигурации

## Использование

### Вариант 1: Запуск вручную (один раз)

```bash
npm start
```

Дайджест будет создан в папке `magazines/` с именем `YYYY-MM-DD.html`

**Результат:**
```
✅ Дайджест сохранён: /path/to/magazines/2024-04-21.html
🌐 Откройте в браузере: file:///path/to/magazines/2024-04-21.html
```

Просто откройте этот файл в браузере! 🎉

### Вариант 2: Автоматический запуск каждый день в 10:00

```bash
npm run schedule
```

Это запустит планировщик, который будет генерировать дайджест каждое утро в 10:00 по Москве.

**Для постоянного запуска (24/7):**

На Windows используйте **Task Scheduler** или **NSSM**:
```bash
# Установите NSSM (http://nssm.cc/)
nssm install fomo-digest "node C:\path\to\schedule.js"
nssm start fomo-digest
```

На Mac/Linux используйте **crontab**:
```bash
crontab -e

# Добавьте строку (10:00 по Москве = 07:00 UTC летом / 08:00 зимой):
0 10 * * * cd /home/username/fomo-digest && node schedule.js >> logs.txt 2>&1
```

### Вариант 3: Ручное обновление во время работы планировщика

```bash
npm run start
```

Это создаст дайджест прямо сейчас, не дожидаясь 10:00.

## Настройка

### Изменить время запуска

В файле `schedule.js` найдите строку:
```javascript
const task = cron.schedule('0 10 * * *', () => {
```

Замените время (в формате `ЧЧ ММ`):
- `0 10 * * *` → 10:00
- `0 9 * * *` → 09:00
- `30 8 * * *` → 08:30

### Добавить свой источник

В файле `digest.js` найдите массив `SOURCES` и добавьте свой:
```javascript
const SOURCES = [
  // ... существующие источники ...
  { url: 'https://ваш-сайт.com/', name: 'Мой блог' }
];
```

### Изменить ключевые слова релевантности

В файле `digest.js` найдите массив `KEYWORDS`:
```javascript
const KEYWORDS = [
  'ai', 'tool', 'claude', // ваши ключевые слова
  'мой-интерес'
];
```

## Способы просмотра дайджеста

### 1. Локально в браузере


- Залейте `digest.js` на сервер
- Настройте cron на сервере
- Опционально: выложите на GitHub Pages / Netlify

## Возможные проблемы

### ❌ "Cannot find module 'axios'"
**Решение:**
```bash
npm install
```

### ❌ "No articles found"
**Причины:**
- Интернет отключен
- Сайт заблокирован вашим провайдером
- Сайт изменил структуру HTML

**Решение:** Проверьте интернет, попробуйте открыть сайт в браузере

### ❌ Планировщик не запускается в фоне
**Решение:** Используйте PM2 (для Linux/Mac) или NSSM (для Windows)

```bash
# Установка PM2
npm install -g pm2

# Запуск в фоне
pm2 start schedule.js --name "fomo-digest"

# Просмотр логов
pm2 logs fomo-digest
```
